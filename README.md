# Solar Data Analysis Project

# Project Setup
## Prerequisites
- Git
- Python 3.13.3

## Installation setups
- Create a virtual environment
  ```
  python -m venv .venv
  ```
- Activate it
  - Windows: 
    ```
    .venv\Scripts\activate
    ```
  - Mac/Linux: 
    ```
    source .venv/bin/activate
    ```
- Install dependencies
  ```
  pip install -r requirements.txt
  ```
- Verify setup
  ```
  pip list
  ```
